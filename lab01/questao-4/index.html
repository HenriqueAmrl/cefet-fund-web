Questão 4

Desafio:

Criar um programa capaz de imprimir todos os números da sequência de Fibonacci de 1 a 100.

Obs.: No código em execução no navegador, a sequência é calculada somente até a 10ª posição, para não travar a página.

Código:

function fib(posicao) {
    switch(posicao) {
        case 1:
            return 1
        case 2:
            return 2
        default:
            return fib(posicao - 1) + fib(posicao - 2)
    }
}

for(let i = 1; i <= 100; ++i) {
    console.log(`Fibonacci na posição ${i} = ${fib(i)}`)
}

*Para ver o código acima funcionando, abra o console.

Questão 5

Desafio:

Criar um programa capaz de imprimir todos os números primos até 1000.

Código:

function primo(num) {
    for(let i = 2; i < num; ++i) {
        if(num % i == 0) return false
    }
    return true
}

for(let i = 2; i <= 1000; ++i) {
    if(primo(i)) {
        console.log(i)
    }
}

*Para ver o código acima funcionando, abra o console.
